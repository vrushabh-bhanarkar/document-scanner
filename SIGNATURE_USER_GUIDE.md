# Signature Feature - User Guide

## üìù How to Add a Signature to Your PDF

### Step 1: Access the Signature Feature

**From Edit PDF Tab:**
1. Open the app
2. Navigate to the **"Edit PDF"** tab at the bottom
3. Scroll through the PDF tools grid
4. Find and tap the **"Add Signature"** card (purple icon with a pen/draw symbol)

**From PDF Editor:**
1. Open any PDF document in the viewer
2. Tap the **"Edit"** button
3. In the toolbar, tap the **"Signature"** button

## ‚úçÔ∏è Method 1: Draw Your Signature

### Draw Tab Features:
- **White Canvas**: A clean drawing area for your signature
- **Color Selection**: Choose between Black, Blue, or Red
- **Smooth Drawing**: Responsive touch drawing
- **Clear Button**: Start over if needed

### How to Draw:
1. Select the **"Draw"** tab at the top
2. Choose your preferred pen color (tap the colored circles)
3. Draw your signature using your finger or stylus
4. If you make a mistake, tap **"Clear"** and try again
5. When satisfied, tap **"Add to PDF"**

**Tips for Best Results:**
- Draw slowly for smoother lines
- Use landscape orientation for larger signatures
- Practice a few times using the Clear button
- Black color is most professional for formal documents

## üì∏ Method 2: Upload a Signature Image

### Upload Tab Features:
- **Gallery Selection**: Choose from existing photos
- **Camera Capture**: Take a new photo of your signature
- **Image Preview**: See how it looks before adding
- **Replace Option**: Easily change your selection

### How to Upload:

#### Option A: From Gallery
1. Select the **"Upload"** tab at the top
2. Tap **"Choose from Gallery"**
3. Navigate to your image

5. Preview the image
6. Tap **"Add to PDF"**

1. Select the **"Upload"** tab at the top
2. Tap **"Take Photo"**

4. Capture the photo
5. Preview the image
6. Tap **"Add to PDF"**

**Tips for Best Photo Signatures:**
- Sign on white paper with a dark pen
- Ensure good lighting
- Take photo directly from above
- Crop out background before uploading
- Use high contrast for better quality

## üîß Troubleshooting

### "Can't draw on the canvas"
- Check if you're on the Draw tab
- Try clearing and starting fresh
- Ensure screen isn't locked

### "Image won't upload"
- Check file format (JPG, PNG supported)
- Ensure image isn't corrupted
- Try taking a new photo
- Check storage permissions

### "Signature appears blurry"
- Use higher quality source image
- Increase signature size on placement screen
- Ensure PDF is high resolution
- Try drawing instead of uploading

### "Can't position signature"
- Make sure to drag from the signature itself
- Check that you're not at screen edge
- Try zooming out if needed

### "Save button not working"
- Ensure signature is within bounds
- Check storage space on device
- Try repositioning signature
- Restart app if issue persists

## ‚ùì Frequently Asked Questions

**Q: Can I use the same signature multiple times?**
A: Yes! Each time you add a signature, you can draw or upload it fresh. For frequently used signatures, save an image to your gallery and upload it.

**Q: Can I edit a signature after adding it?**
A: No, once saved to PDF, it's permanent. You'll need to use the original PDF and add the signature again.

**Q: Does this create a legally binding signature?**
A: This creates a graphical representation of your signature. Legal validity depends on your jurisdiction and document type.

**Q: Can I add multiple signatures to one document?**
A: Yes! Open the signed PDF and add another signature. Each creates a new version.

**Q: What file formats are supported?**
A: Currently only PDF files. The output is also PDF format.

**Q: Is there a signature limit?**
A: No limit on the number of signatures you can add.
